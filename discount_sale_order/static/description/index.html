Odoo Discount On Sale Order

The module adds new discount types in Odoo Sales order which are not present by default. We have added a FIXED DISCOUNT Type to the sales order line. We have also introduced a GLOBAL DISCOUNT CATEGORY which can be fixed or percentage type and apply on the entire sales order amount.

Information

https://webkul.com/blog/odoo-discount-sale-order/

Note: The module is dependent upon the Odoo Discounts on Invoices and Bills. You need to install the module Odoo Discounts on Invoices and Bills for the module to work properly.

While Odoo Discounts on Invoices and Bills module gives discount options while creating invoices and bills, this module provides options to set various discounts on Sales Orders in Odoo.

Why Do You Need This Module?

Because Odoo Only allows percent discount in sales order lines

  • When it comes to discounts you should have more flexibility and options to apply different discounts.
  • What if you have multiple products in a sales order; you cannot apply an order line discount to each product. You want to have an option for adding a global discount.

This is what Odoo discount on Sales Order provides you with

  • Manage discounts Types- fixed or percent.
  • Flexibility to choose the type of discount category- order line or global discount.

Complete Control Over Discounts On Your Sales Order

  • From sales order To Invoices, the Odoo Discount modules provide multiple options to apply discounts in Odoo.
  • Set different types of discounts are different levels in a Odoo sales order.

Apply Fixed/Flat Discount Type On Sales Order Lines In Odoo

Default Functionality

  • Odoo Only allows you to add percentage discount Types in Order lines of sales order.

With Our Module

  • You can now add a Fixed or Flat Discount type in the order lines of the sales order.
  • You can set the new price of the product in the pricelist and the module shows the difference of original and new sale price as fixed discount on sale order line.
  • The amount can be taxed or untaxed.

Introducing Global Discount Category Applicable on the Entire Sale Order

Note: By default, you can not add any global discount i.e. discount on the entire sales order amount.

With our module

  • Global discount is applied on the total sale order amount.
  • The module allows you to add fixed or percent discount type on the whole sales order.
  • Again the discount can be given on taxed and untaxed sales order amount as set in the settings.

Summarized Features List

  • Apply fixed and percent type discount on Odoo Sales order lines.
  • You can also apply a fixed or percentage discount to the sale order line using the pricelist. (see screenshot below to know how)
  • Apply Fixed and percent type Global discount on the entire sales order directly from the sales order form.
  • Apply discounts on taxed and untaxed amount (easily set from configuration settings).
  • Import the applied discount to the sales order reports.
  • Manage Separate Account for Global Discount from the settings.
  • The discounts applied on sales orders are also carried automatically forward to the generated invoices with the help of base module - Odoo Discounts On Invoices and Bills.
  • You can view all the discount types applied in a sales order in the sales order preview.
  • View the applied discount types and amounts in the printed/downloaded sale order.
